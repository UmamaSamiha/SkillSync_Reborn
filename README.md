# SkillSync

A collaborative academic tracking platform built with Flask and React. SkillSync helps students track progress, manage assignments, stay focused, and grow together — while giving admins powerful tools to monitor engagement, detect issues, and support student success.

---

## Features

### Content Integrity & Edit Tracking
Admins can track every edit made to submissions, detect sudden large changes (such as AI-generated paste), view full edit timelines, and compare versions side by side.

### Question Bank & Resource Management
The system organizes question banks and learning resources with enforced difficulty levels, ensuring students progress through material in a structured way.

### Prerequisite Enforcement
Students must demonstrate mastery of a topic before unlocking the next one. The system enforces prerequisite chains to maintain learning progression.

### Student Portfolio
The system automatically builds a student portfolio showcasing projects, best assignments, acquired skills, and contribution statistics.

### Time Tracking & Productivity Analytics
A built-in time-tracking engine provides weekly productivity reports and per-topic analytics so students can see where their time goes.

### Contribution Analysis
For group projects, tasks are analyzed to show how much each member contributed, promoting accountability and fair grading.

### Assignment Submission & File Upload
Students can submit assignments and upload files directly through the platform.

### Deadline Notifications
Users receive notifications for upcoming deadlines and completed tasks to stay on top of their workload.

### AI & Similarity Detection
Admins can scan submissions for AI-generated content and check for similarities between student work, with detection scores stored for review.

### Collaboration Heatmap
The system generates a collaboration heatmap highlighting inactive members and visualizing contribution distribution across teams.

### Focus Mode (Pomodoro) & Session History
Students can work in focus mode using a built-in Pomodoro timer and track their session history over time.

### Engagement Scoring
The system calculates an engagement score for each student based on forum activity, submissions, resource access, and quiz attempts.

### Risk Detection System
The platform flags students showing signs of risk — low attendance, falling grades, and late submission patterns — with automated alerts for admins.

### Performance Trends & Grade Prediction
Performance trend graphs and final grade predictions help both students and admins understand academic trajectory.

### Student Classification & Personalized Feedback
The system classifies students as consistent performers or at-risk and helps personalize feedback based on their patterns.

### Certificate Generation
The platform generates certificates that include grades, study hours, and a digital verification code for authenticity.

## Tech Stack

- **Backend:** Python, Flask, PostgreSQL, JWT Authentication
- **Frontend:** React, Vite, React Router, Axios
